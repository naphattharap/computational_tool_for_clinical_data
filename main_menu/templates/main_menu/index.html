Application

My inspiration for this application

Building Machine Learning Models is a trial and error, and some steps are like a set of pattern of tasks ex. statistical analysis, correlation of attributes, etc.

Isn't it better to summarize those basic steps and visualize data quickly through the FREE software?
My intention to develop this software is to support tasks in Machine Learning Pipeline.
The ultimate goal is to do automatic model selection (it's still long way to go...)

I have a special interest in applying Artificial Intelligence in Healthcare domain.
Therefore, I developed The computational tool for finding an association between radiomics data and clinical outcomes for my master thesis in AI.
The tool is integrated to this platform for sample work in applying machine learning.
