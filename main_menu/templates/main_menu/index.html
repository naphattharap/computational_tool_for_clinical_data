My inspiration for this application

    As we have known that to build Machine Learning models, it is trial and error.
Instead of modifying, writing similar source code or doing the same kind of tasks again and again, I am summarizing basic steps in Machine Learning and developing this application to help Machine Learning Engineers or Data Scientists do some basic tasks e.g. basic analysis data, visualize data, model selection pipeline etc.

Here is an example that I used this tool for finding features based on target variable.
It is customized for selecting radiomics features based on selected clinical outcomes
For more details, please visit my master's thesis at https://repositori.upf.edu/handle/10230/42547
