Sample work for applied machine learning in real-world problem

The computational tool for finding an association between radiomics data and clinical outcomes
I integrated my work for my master's thesis into this application.
It is the computational tool for finding an association between radiomics data and clinical outcomes.
The dataset are cardiovascular radimics data which comes with 8xx features and 10,xxx columns of clinical outcomes data.
Since the dataset are large and it is not easy to find association of those data.
To solve this problem, I implemented the tool for finding related features based on selected target in clinical outcomes so that a clinician can see what features are related to frequency of hard exercise, smoking status, age, gender and so on. In addition, Framingham Risk Score for cardiovascular disease is also calculated for selected the input dataset.
